# Контекст проекта — КАДРЫ (HRMS)

## Обзор каталога

Этот каталог содержит документацию для мультитенантной системы управления персоналом (HRMS) холдинга из 8 компаний. Система автоматизирует работу кадрового отдела: ведение личных дел, учёт посещаемости, расчёт зарплат, управление инвентарём.

## Ключевые файлы

- **`HRMS_TZ.md`** — актуальное техническое задание с описанием всех реализованных модулей, ролей, архитектуры и API. **Основной справочный документ.**
- **`ТехЗадание.md`** — исходный документ ТЗ (ранняя версия, MVP-требования).
